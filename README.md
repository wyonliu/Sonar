# Sonar · 声呐

> **在你找到另一个人之前，先找到爱里真实的自己。**

---

## 什么是 Sonar

声呐的工作原理：发出你听不见的波，探测水面下看不见的东西，把隐藏的轮廓呈现出来。

**Sonar 做同样的事——只是探测的对象是你在关系里的真实样子。**

大多数人谈了一段又一段恋爱，却从未真正认识自己在关系里是什么人。你以为你是安全型，其实你用独立在保护自己。你以为你需要温柔，其实你需要一个能hold住你的人。

Sonar 通过**沉浸式多媒体测试**，整合 7 大心理学与人类学体系，生成你独一无二的**关系星图**——揭示你的关系模式、你的核心渴望、以及你与他人之间真正的共鸣频率。

这不是一个约会 App。这是**人类第一个关系自我认知平台**。

---

## 核心体验

```
观看真实的视频场景  →  做出本能选择  →  AI 多维分析
        ↓
生成你的关系星图（7 个维度，独一无二）
        ↓
揭示你的「关系阴影」：你不知道的自己
        ↓
发现与你真正共鸣的人
```

---

## 产品亮点

- **7 维收敛测量**：神经科学 + 依恋理论 + Gottman 研究 + 九型人格 + 八字 + 荣格原型 + 叙事心理学
- **场景视频测试**：不是填表，是沉浸式体验，绕过自我报告偏差
- **关系阴影揭示**：展示自我认知与实际行为的落差
- **关系星图**：极美的可视化，天然传播
- **双人模式**：情侣/朋友比较，共鸣图生成
- **AI 月老**：不是算法推人，是 AI 替你讲故事

## 快速开始

```bash
# 克隆项目
git clone git@github.com:wyonliu/Sonar.git
cd Sonar

# 安装依赖
pnpm install

# 启动开发环境
pnpm dev
```

---

## 技术栈

- **App**: React Native (Expo)
- **Web**: Next.js 14
- **Backend**: FastAPI (Python)
- **AI**: Claude API + 自研匹配模型
- **数据库**: PostgreSQL + Redis
- **媒体**: Cloudflare R2 + Stream
- **实时**: WebSocket
